Exercice JS autovalidant :

exercice ci dessous a pour objectif de vous familiariser avec la programmation orientee objet en javascript.
dans ce exercice, vous devez creer des fonction pour améliorer le fichier index.html afin de créer une petit calculette de base.


Completter la fonction "calculerSurface()" dans le scripts.js pour pouvoir calculer les information manquantes dans le fichier index.html (surface total, nb carreaux...) 


Pour tester si votre fonction renvoie de bonne valeur lancer dans la console la commande suivante :
test()
